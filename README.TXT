TITULO DO PROJETO: Trabalho 2 - Datas

OBJETIVO DO PROJETO: 
Neste trabalho desenvolvemos um código com os objetivos de descobrir a idade em anos, meses e dias, o signo, quantos dias faltam
para o aniversário e o dia da semana que nasceu do usuário.

VERSÃO ou DATA: OFICIAL - 20210427

COMO INICIAR O PROJETO: 
É necessário executar o programa e escolher uma das opções informadas pelo menu. E em seguida, seguir os passos exigidos pelo programa.

AUTORES: Tomaz Bettio e Eduardo Balzan

INSTRUÇÕES PARA O USUÁRIO:
É necessário executar o programa e escolher uma das opções informadas pelo menu. E em seguida, seguir os passos exigidos pelo programa.

DIFICULDADES:
Durante a realização deste trabalho foi percebido algumas dificulades pela dupla. Sendo algumas delas sobre as utilizações de laços
(for, do while, while) e também na lógica que resolvesse qualquer valor informado pelo usuário. 

O QUE FOI UTILIZADO:
Utilizamos as bibliotecas do java denominadas "Calendar" e "Scanner". Sendo que, uma das maiores vantagens de utilizar a biblioteca "Calendar" é o fato do cálculo
do número de dias que cada mês possui já estar definido automaticamente.
